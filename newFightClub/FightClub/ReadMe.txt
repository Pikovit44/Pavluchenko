1) Загружается форма MainForm
2) В MainForm создается presenter с интерфейсом (IMainForm) этой формы в качестве параметра для конструктора. 
   (в presenter создаются экземпляры player, bot, controller, textInfo) 
3) Пользователь устанавливает настройки боя
4) По нажатию кнопки "Бой" проверяется что выбрал на форме пользователь и срабатывает событие FightClick
5) На событие FightClick реагирует presenter и запускает методы:
	SetupPosition (установка частей тела для блока/удара из MainForm в свойства Player'a и Bot'a)
	Вызывается метод controller.Fight() из класса GameController , в котором срабатывают методы GetHit() и Setblock() у экземпляров игрока и бота
	Вызывается метод View() который передает в форму изменения которые произошли во время боя
6) Исходя из результатов методов GetHit() и Setblock() возникают события Wound, Block или Death. 
7) На эти события подписан экземпляр textInfo класса TextInfo, который записывает в лог из MainMenu информацию по каждому раунду
